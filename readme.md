# Morphological analysis

お試し形態素解析 with kuromoji.js

## Usage

```
# Install dependencies
$ yarn

# Generate funny sentences
$ node index.js
```

## Reference

[kuromoji.jsをつかった形態素解析](https://zeny.io/blog/2016/06/16/kuromoji-js/)
